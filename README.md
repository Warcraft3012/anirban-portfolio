# Anirban Chakravertty's Portfolio

Welcome to the portfolio website of Anirban Chakravertty, a passionate Data Analyst and Backend Developer. This project showcases my skills, experience, and projects in a visually engaging format inspired by the retro TRON aesthetic.

## Project Structure

The portfolio is organized into the following structure:

```
anirban-portfolio
├── src
│   ├── index.html          # Main HTML file for the portfolio
│   ├── css
│   │   ├── styles.css      # Main styles for layout and typography
│   │   ├── animations.css   # Keyframe animations and transitions
│   │   └── tron-theme.css   # Styles for the retro TRON aesthetic
│   ├── js
│   │   ├── main.js         # Main JavaScript for initialization and interactions
│   │   ├── animations.js    # Functions for triggering animations
│   │   └── components.js     # Reusable components for features
│   ├── assets
│   │   ├── fonts           # Custom fonts for enhanced typography
│   │   ├── videos          # Background videos and animations
│   │   └── data.json       # Structured data for projects and skills
│   └── sections
│       ├── hero.html       # Hero section with name and tagline
│       ├── skills.html     # Dynamic summary of skills
│       ├── projects.html   # Project gallery with interactive cards
│       ├── experience.html  # Experience timeline with animated flow
│       ├── education.html   # Education and certifications section
│       └── contact.html     # Contact information and links
├── package.json             # npm configuration file
├── .gitignore               # Files to be ignored by version control
└── README.md                # Project documentation
```

## Features

- **Hero Banner**: A captivating introduction to Anirban with contact information and a personal tagline.
- **Skills Summary**: A dynamic display of technical skills including Python, SQL, Power BI, Streamlit, Go, and cloud platforms.
- **Project Gallery**: An interactive showcase of projects with hover effects and concise descriptions.
- **Experience Timeline**: A visually engaging timeline highlighting professional experience with animated elements.
- **Education and Certifications**: A modern display of educational background and certifications.
- **Skills Grid**: A grid layout showcasing various skills with visual indicators.
- **Contact Section**: Links to download the resume and connect on LinkedIn, along with contact details.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd anirban-portfolio
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `src/index.html` in your browser to view the portfolio.
